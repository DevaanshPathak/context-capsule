Sample Research Note

Native messaging lets a browser extension pass page context to a local process. Context Capsule uses that bridge to capture the active page title, URL, selected text, and clipboard fallback, then stores a recoverable history entry after copying a clean markdown block.

  • With selected text, the pasted block should contain this page title, this local URL, and only the selection.
  • With no selected text, the pasted block should use whatever was already on your clipboard.
  • The popup history should show this capture and let you copy, pin, delete, or clear it.
